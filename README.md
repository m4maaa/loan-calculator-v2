# ระบบประเมินเงินกู้ อทบ. v2

โปรเจกต์นี้เป็นเวอร์ชันยกเครื่องใหม่ของระบบคำนวณเงินกู้ อทบ. แบบ static web app เปิดไฟล์ `index.html` ได้ทันที หรือรันด้วย Live Server ใน VS Code ก็ได้

## จุดเด่นของเวอร์ชันนี้

- คำนวณเงินเดือนอัตโนมัติจาก **ระดับ + ชั้นเงินเดือน**
- สลับไปใช้ **เงินเดือนกรอกเอง** ได้
- คำนวณค่างวดอัตโนมัติจาก **วงเงินกู้ + ดอกเบี้ย + จำนวนงวด**
- รองรับทั้ง **กู้เฉพาะเงินฝาก** และ **กู้เกินเงินฝาก**
- สร้างฟอร์ม **ผู้ค้ำประกันแบบไดนามิก** ตามยอดเกินเงินฝาก
- ติ๊กสถานะว่าผู้ค้ำ **ผ่านเกณฑ์ 1 ใน 3** ได้
- ประเมินผล **ผ่าน / ควรปรับก่อนยื่น / ไม่ผ่าน**
- แสดง **ผลตรวจรายข้อ** และ **คำแนะนำ**
- เปรียบเทียบหลายจำนวนงวดอัตโนมัติ
- มี **ประวัติการคำนวณ**, **บันทึกร่าง**, **Export JSON**, และรองรับพิมพ์เป็น PDF

## โครงสร้างไฟล์

- `index.html` หน้าเว็บหลัก
- `styles/main.css` สไตล์ทั้งหมด
- `data/config.js` ค่ากฎเกณฑ์และตัวเลือก
- `data/salary.js` ตารางระดับ/ชั้นเงินเดือน
- `js/utils.js` ฟังก์ชันช่วยเหลือ
- `js/calculator.js` เครื่องคำนวณและ rules engine
- `js/ui.js` งาน render หน้าจอ
- `js/app.js` ตัวควบคุมหลักของแอป

## วิธีทดสอบใน VS Code

1. เปิดโฟลเดอร์นี้ใน VS Code
2. ถ้ามี Live Server ให้คลิก `Go Live`
3. หรือเปิดไฟล์ `index.html` ตรงในเบราว์เซอร์ก็ได้

## จุดที่ควรตรวจสอบก่อนใช้จริง

ไฟล์ `data/salary.js` ถูกออกแบบให้แก้ไขง่ายมาก โดยตอนนี้ใส่ **ตารางตั้งต้นแบบพร้อมใช้งาน** ไว้แล้วเพื่อทดสอบระบบ end-to-end แต่ถ้าจะนำไปใช้หน้างานจริง ควรตรวจสอบและอัปเดตตัวเลขเงินเดือนแต่ละระดับ/ขั้นให้ตรงกับประกาศ/บัญชีอัตราปัจจุบันของหน่วยงานอีกครั้ง

## จุดที่แก้กติกาได้ง่าย

- เกณฑ์ 1 ใน 3 และขั้นต่ำ 5,000 บาท: `data/config.js`
- ช่วงยอดเกินเงินฝากที่ใช้ผู้ค้ำ 1 / 2 / 3 คน: `data/config.js`
- จำนวนงวดที่อนุญาต: `data/config.js`
- ตารางเงินเดือน: `data/salary.js`



## อัปเดตตารางเงินเดือนจาก Excel
ตอนนี้ระบบรองรับจำนวนชั้นเงินเดือนของแต่ละระดับที่ไม่เท่ากันแล้ว และ `data/salary.js` ถูกสร้างจากไฟล์ Excel ได้โดยตรง

ตัวอย่างคำสั่ง:
```bash
python tools/generate_salary_js.py "ตารางเงินเดือน ฉบับใหม่.xlsx"
```

ถ้าต้องการระบุปลายทางเอง:
```bash
python tools/generate_salary_js.py "ตารางเงินเดือน ฉบับใหม่.xlsx" "data/salary.js"
```

รูปแบบไฟล์ Excel ที่รองรับ:
- แถวแรกเป็นชื่อระดับ เช่น `ป.1`, `ป.2`, `ป.3`, `น.1`, `น.2`, `น.3`, `น.4`
- แต่ละระดับใช้ 2 คอลัมน์ติดกัน คือ `ชั้น` และ `เงินเดือน`
- จำนวนชั้นของแต่ละระดับยาวไม่เท่ากันได้
- รองรับค่าแบบ `0.5/เยียวยา`
